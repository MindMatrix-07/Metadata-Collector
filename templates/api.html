How to get Spotify API Keys?
  1. Go to the Spotify Developer Dashboard and log in.
  2. Click on "Create app".
  3. Fill in the App name and description (e.g., "Metadata Collector local").
  4. Set the Redirect URI to http://localhost:5000 or your actual domain.
  5. Select "Web API" under APIs used.
  6. Agree to the ToS and save.
  7. Click "Settings" in your new app to view your Client ID and Client Secret.
